

## Schemas

- autogenerated dart files json_schema.dart and ui_schema.dart
- generated with quicktype `npm install -g quicktype`
`quicktype -s schema lib/src/schemas/json-schema_draft7.json -o lib/src/models/json_schema.dart`
`quicktype -s schema lib/src/schemas/ui/ui.schema.json -o lib/src/models/ui_schema.dart`
